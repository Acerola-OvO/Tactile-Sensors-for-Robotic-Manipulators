# Sensor fabrication materials

The flexible sensing layer was fabricated from commercially available materials. Quantities and prices vary by supplier and are not treated as a verified project cost estimate.

| Material | Role |
|---|---|
| Velostat / Linqstat sheet | Piezoresistive sensing layer |
| Polyimide film | Flexible protective and support layers |
| 3M 468MP adhesive transfer tape | Layer bonding and electrode fixation |
| Conductive thread / fine conductive wire | Orthogonal row and column electrodes |
| Flexible flat cable and FPC connectors | Sensor-to-board interconnection |
| Arduino Nano | ADC sampling and serial transmission |
| CD74HC4067 | Selection of 16 sensor rows |
| 2 × SN74HC595 | Cascaded selection of 16 sensor columns |
| 20 kΩ resistor | Voltage-divider reference / pull-down |

This list was prepared from the materials used in the prototype and the reference 3D-ViTac material list. It should be replaced with purchase records before making a precise cost claim.
